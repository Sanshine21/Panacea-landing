## 1. Что уже сделано

### 1.1 Страница (index.html)
Сайт представляет собой одностраничный лендинг из 7 секций. Прокрутка между ними плавная (при клике на пункт меню страница плавно едет к нужному блоку).

| Секция | Что внутри | Статус |
|--------|-----------|--------|
| **Hero** (главный экран) | Заголовок «Панацея», описание проекта, две кнопки («Смотреть видео», «Записаться»), иконки Telegram и YouTube, фоновая фотография | Готово |
| **О проекте** | Текст о проекте, список руководителей (Олег — с фото, Артём и Валерия — аватарки с инициалами), заглушка для фото/видео | Готово |
| **Видео** | 3 встроенных ролика с YouTube в карточках | Готово |
| **Услуги** | 4 карточки: подбор центра (бесплатно), подбор специалиста (бесплатно), обзор и партнёрство (по договорённости), консультация (6000 ₽) | Готово |
| **Запись на консультацию** | Информация о цене + форма заявки | Готово |
| **Бесплатная помощь** | 3 карточки: подбор центра, подбор специалиста, горячие линии | Готово |
| **Контакты** | Email, Telegram, YouTube, телефон | Готово |
| **Подвал (footer)** | Копирайт | Готово |

### 1.2 Меню
- Фиксированная панель сверху (остаётся при прокрутке).
- Логотип слева (картинка `Лого 4 чб.png`).
- 6 пунктов: О проекте, Видео, Услуги, Запись, Бесплатная помощь, Контакты.
- На мобильном превращается в «гамбургер» (три полоски), при нажатии раскрывается список.

### 1.3 Форма заявки
- Отправляет данные через сервис FormSubmit на почту `inndxx@gmail.com`.
- Поля: имя (обязательно), телефон (обязательно), email, тема обращения (выпадающий список: Для себя / Для близкого / Другое), сообщение.
- После отправки человек видит стандартное сообщение от FormSubmit.

### 1.4 Анимации
- Текст на главном экране появляется постепенно (снизу вверх) при загрузке страницы.
- Остальные секции «проявляются» по мере прокрутки — блок плавно выезжает снизу и становится видимым.
- При наведении на карточки (услуги, видео, помощь) они слегка приподнимаются и подсвечиваются.

### 1.5 Мобильная версия
- На экранах уже 768 пикселей меню превращается в гамбургер.
- Текст и кнопки перестраиваются в одну колонку.
- Карточки тоже выстраиваются друг под другом.

## 2. Фактический дизайн

В PRD изначально планировалась светлая тема с цветами графит/teal. В процессе вайб-кодинга вышла тёмная тема с золотым акцентом.

| Элемент | Фактическое значение | Где используется |
|---------|---------------------|-----------------|
| Основной фон | `#0B0D10` (чёрно-синий) | Вся страница |
| Карточки / блоки | `#12151C` (тёмно-синий) | Фон секций, форма, карточки |
| Акцентный цвет | `#D4A853` (золотой) | Кнопки, ссылки, иконки, цены |
| Светлый акцент | `#E8C989` (светло-золотой) | Обводка кнопок, подсветка |
| Основной текст | `#F0EBE3` (тёплый белый) | Заголовки, параграфы |
| Вторичный текст | `#8A8F99` (серый) | Подзаголовки, примечания |
| Рамки | `#2A2F3A` (тёмно-серый) | Границы карточек |
| Шрифт заголовков | Playfair Display | Все h1, h2, h3 |
| Шрифт основного текста | Manrope | Параграфы, кнопки, формы |

## 6. Как запустить локально

1. Открыть папку `site-oleg` в редакторе.
2. Запустить простой сервер (чтобы сайт работал корректно):
   ```bash
   python -m http.server 8000
   ```
3. Открыть в браузере: `http://localhost:8000`

Или просто открыть файл `index.html` двойным кликом (но некоторые элементы могут работать с ограничениями из-за политик безопасности браузера).
