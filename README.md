# Panacea Landing

## Название проекта

**«Панацея — Лендинг»**

Одностраничный сайт-визитка (лендинг) для видео-проекта «Панацея», посвящённого помощи в преодолении зависимостей, психологии и выходу из кризиса.

---

## Какую задачу решает

Главная задача — **соединить человека, столкнувшегося с зависимостью (своей или близкого), с реальной помощью.**

Проект решает проблему информационного вакуума: вокруг зависимости много мифов, навязанных услуг и страха. Лендинг показывает, какая помощь существует на самом деле, знакомит с командой проекта, демонстрирует видео-контент и позволяет оставить заявку на консультацию или подбор специалиста.

---

## Для кого создаётся проект

1. **Человек в зависимости** — ищет выход, но не знает, куда идти и с чего начать.
2. **Родственник или друг (созависимый)** — видит проблему у близкого, переживает, не понимает, как помочь и не сломать.
3. **Тем, кто ищет проверенного психолога** — нужен специалист по зависимостям, а не случайный человек с доски объявлений.
4. **Реабилитационные центры и психологи** — для партнёрства, обзоров, репортажей и сотрудничества.

## Основные функции

| Функция | Описание |
|---------|----------|
| **Главный экран (Hero)** | Большое фоновое изображение с заголовком, описанием проекта и двумя кнопками: «Смотреть видео» и «Записаться на консультацию». |
| **О проекте** | Текстовый блок с описанием миссии + карточки команды (CEO, CTO, режиссёр) + **карусель фотографий** с автопроигрыванием, стрелками и поддержкой свайпа на телефоне. |
| **Видео** | Встроенные ролики с YouTube-канала проекта. Пользователь сразу видит контент, не уходя с сайта. |
| **Услуги** | Четыре карточки: подбор центра (бесплатно), подбор психолога (бесплатно), обзор и партнёрство, консультация с психологом (6 000 ₽, 60 минут). |
| **Кликабельные карточки** | При нажатии на любую карточку услуги страница плавно прокручивается к форме записи. |
| **Форма заявки** | Поля: имя, телефон (автоматическая маска +7), email, тема обращения, сообщение. Отправка через FormSubmit на email проекта. |
| **Страница «Спасибо»** | После отправки формы пользователь видит подтверждение и ссылку на Telegram для срочной связи. |
| **Бесплатная помощь** | Блок с описанием бесплатных услуг: подбор центра, подбор специалиста, горячие линии. |
| **Контакты** | Email, телефон (кликабельный), ссылки на Telegram и YouTube. |
| **Адаптивная вёрстка** | Корректное отображение на компьютере, планшете и телефоне. Бургер-меню на мобильных. |
| **Анимации** | Плавное появление блоков при прокрутке, анимация кнопок и карточек, смена фона шапки при скролле. |
| **SEO-настройки** | Метатеги описания и ключевых слов, Open Graph для соцсетей, микроразметка Schema.org. |

## Форма результата

**Результат работы пользователя с сайтом:**

- Знакомство с проектом через текст, фото и видео.
- Понимание, какие услуги доступны и сколько они стоят.
- Возможность оставить заявку на бесплатную или платную помощь.
- Получение контактов для связи (Telegram, email, телефон).
- После отправки формы — переход на страницу подтверждения с информацией о дальнейших шагах.

**Результат для владельца проекта:**

- Заявки с сайта приходят на указанный email в виде структурированных писем.
- Пользователи видят контент проекта и переходят на YouTube / Telegram.
- Сайт работает как точка входа для новых клиентов и партнёров.

---

## Минимальный функционал

Чтобы сайт выполнял свою задачу, достаточно следующего:

1. **Главный экран** — кто мы и что делаем (2-3 предложения) + кнопка к форме.
2. **Блок услуг** — минимум 2-3 карточки с ценами или пометкой «бесплатно».
3. **Форма заявки** — имя, телефон, тема обращения. Отправка работает.
4. **Контакты** — способ связи (Telegram, email или телефон).
5. **Адаптивность** — сайт читаем на телефоне без горизонтальной прокрутки.

Всё остальное (видео, карусель, анимации, SEO) — это улучшения, которые делают сайт убедительнее, но не обязательны для запуска.

## Технологии

| Технология | Для чего используется |
|------------|----------------------|
| **HTML5** | Разметка страницы: заголовки, секции, формы, ссылки, изображения. |
| **CSS3** | Оформление: цвета, шрифты, отступы, расположение элементов, анимации, адаптивные сетки. |
| **JavaScript** | Интерактивность: бургер-меню, карусель фото, анимации при скролле, маска телефона, кликабельные карточки. |
| **Google Fonts** | Подключение шрифтов Playfair Display (заголовки) и Manrope (текст) из интернета. |
| **Font Awesome 6** | Иконки: стрелки, телефон, почта, соцсети, меню. |
| **FormSubmit** | Сторонний сервис для отправки данных формы на email без собственного сервера. |
| **Git + GitHub** | Контроль версий и хранение кода в облаке. |

**Что НЕ используется:** фреймворки вроде Bootstrap, Tailwind, React, Vue; библиотека jQuery. Весь код написан с нуля.

## Как открыть сайт

**Онлайн:** [https://livepanacea.help](https://livepanacea.help)

**На компьютере:** открой папку проекта и дважды кликни по файлу `index.html`.

**Важно:** форма заявки работает только если сайт открыт через интернет (хостинг). При открытии как локального файла (`file:///...`) отправка на почту не сработает.
