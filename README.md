# Панацея — Лендинг видео-проекта о помощи при зависимостях

## Что это

Одностраничный сайт-визитка (лендинг) для проекта «Панацея». Показывает, чем занимается проект, какие услуги предлагает, и позволяет оставить заявку на консультацию.

**Адрес в интернете:** [livepanacea.help](https://livepanacea.help)

---

## Для кого полезен

- Людям, столкнувшимся с зависимостью (своей или близкого человека).
- Тем, кто ищет проверенного психолога или реабилитационный центр.
- Партнёрам — центрам реабилитации, психологам для сотрудничества.

---

## Что умеет сайт

| Возможность | Что это значит |
|-------------|----------------|
| **Адаптивная вёрстка** | Сайт нормально выглядит и на компьютере, и на телефоне. |
| **Главный экран (Hero)** | Большая картинка с заголовком, кратким описанием и двумя кнопками. |
| **Карусель фото** | В разделе «О проекте» можно листать фотографии стрелками или пальцем на телефоне. |
| **Встроенные видео с YouTube** | На странице сразу видны выпуски проекта, не нужно уходить на канал. |
| **Карточки услуг** | Четыре блока: подбор центра (бесплатно), подбор психолога (бесплатно), партнёрство, платная консультация (6 000 ₽). |
| **Кликабельные карточки** | При нажатии на любую карточку услуги страница плавно прокручивается к форме записи. |
| **Форма заявки** | Поля: имя, телефон (автоматически форматируется как +7), email, тема обращения, сообщение. |
| **Отправка заявки на почту** | Данные из формы уходят на email через сервис FormSubmit. |
| **Страница «Спасибо»** | После отправки формы пользователь попадает на страницу с подтверждением. |
| **Фиксированная шапка** | Меню всегда наверху экрана, даже когда листаешь вниз. Меняет цвет при прокрутке. |
| **Бургер-меню на телефоне** | Три полоски вместо обычного меню — экономит место на маленьком экране. |
| **Анимация появления блоков** | Разделы плавно выезжают при прокрутке страницы. |
| **SEO-настройки** | Поисковые системы (Яндекс, Google) лучше понимают, о чём сайт. Есть теги описания, ключевых слов и разметка для соцсетей. |

## На чём сделан

- **HTML** — язык разметки, который говорит браузеру, где какой блок стоит (заголовок, картинка, форма).
- **CSS** — язык стилей, отвечает за цвета, шрифты, расположение элементов и анимации.
- **JavaScript** — язык программирования, добавляет интерактивность: карусель, меню, маску телефона, анимации при скролле.
- **Google Fonts** — шрифты загружаются из интернета, а не хранятся в папке.
- **Font Awesome** — библиотека иконок (стрелки, телефон, почта, соцсети).
- **FormSubmit** — сторонний сервис, который принимает данные из формы и отправляет их на указанный email.

**Без фреймворков** — не используется Bootstrap, Tailwind, jQuery и другие готовые библиотеки. Всё написано с нуля.

## Структура папок

```
site-oleg/
├── index.html          ← Главная страница (лендинг)
├── thanks.html         ← Страница «Спасибо» после отправки формы
├── css/
│   └── style.css       ← Все стили сайта (цвета, шрифты, расположение)
├── js/
│   └── main.js         ← Скрипты (меню, карусель, анимации, маска телефона)
├── assets/
│   └── images/         ← Все картинки: логотип, фото команды, фоновые изображения
└── favicon.png         ← Иконка сайта во вкладке браузера
```

## Как запустить на компьютере

1. Открой папку проекта.
2. Дважды кликни по файлу `index.html` — он откроется в браузере.
3. Или, если используешь редактор **ZED**, можно открыть файл и запустить через Live Server (если установлен).

**Важно:** форма заявки работает только если сайт открыт через интернет (хостинг). Если открывать просто как файл с компьютера, заявки отправляться не будут.

---

## Как выложить в интернет

Сайт уже развёрнут на домене `livepanacea.help`. Для обновления нужно загрузить изменённые файлы на хостинг (сервер, где лежит сайт). Способ зависит от того, какой хостинг используется (FTP, панель управления, GitHub Pages, Netlify и т.д.).
